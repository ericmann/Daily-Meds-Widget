== Description ==

Adds a sidebar widget with the latest mediation from DailyMedToday.com.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire `daily-meds-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How often do I get new meditations? =

The plugin will automatically check DailyMedToday.com for new meditations once every 6 hours.  If a new meditation has
been posted, you'll get it immediately.  If not, you won't see anything.
